# Modelado de Objetos en Java: Clase TV 📺

Este repositorio contiene el desarrollo de la **Tarea 2.1** de la materia Programación I (P1). El objetivo principal del proyecto es aplicar los conceptos fundamentales de la **Programación Orientada a Objetos (POO)** en Java, tales como la abstracción, encapsulamiento, definición de atributos, métodos y la instanciación de objetos.

El proyecto fue desarrollado estructuradamente en **NetBeans** utilizando el sistema de construcción de **Apache Ant**.

---

## 📁 Estructura del Código

El código fuente se encuentra dentro del directorio `src/` y está compuesto por dos clases principales:

*   **`Tv.java`**: Representa la clase molde (entidad). Define el estado y comportamiento de un televisor mediante atributos (como canal, volumen, estado de encendido/apagado) y métodos para manipular dicho estado de forma segura.
*   **`Prueba.java`**: Es la clase que contiene el método principal (`main`). Se encarga de instanciar diferentes objetos de la clase `Tv`, simular interacciones del usuario (encender, cambiar de canal, subir volumen) y mostrar los resultados en la consola.

---

## 🛠️ Conceptos de POO Demostrados

*   **Abstracción:** Modelado de un objeto del mundo real (un televisor) en código de software.
*   **Encapsulamiento:** Control del estado interno de los objetos a través de métodos específicos.
*   **Instanciación:** Creación de múltiples objetos independientes basados en una misma clase base.
